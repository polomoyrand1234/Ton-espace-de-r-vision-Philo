Le langage

Parler : communiquer, penser, nommer. Le langage révèle-t-il ou déforme-t-il le réel ?

Notions
Signe • Sens • Référence
Axes
Pensée • Communication • Vérité
Auteurs
Saussure • Wittgenstein • Platon

Définitions

  • Langage : système de signes pour exprimer/communiquer.
  • Signe : signifiant (forme) + signifié (idée).
  • Langue : système commun (règles), différent de la parole (usage individuel).

Astuce bac : langage = outil de pensée + outil social.

Problèmes

Peut-on penser sans langage ?
Débat : langage structure la pensée, mais il existe des formes non verbales.
Les mots disent-ils le réel ?
Ils classent et simplifient : utile, mais parfois trompeur.
Le langage peut-il manipuler ?
Oui : rhétorique, propagande, ambiguïtés.

Auteurs

Saussure

Le signe est arbitraire : le lien mot/chose n’est pas naturel.

➡️ signe

Wittgenstein

Sens = usage : comprendre un mot, c’est savoir l’employer.

➡️ usage

Platon

Critique de la rhétorique : parler peut séduire sans vérité.

➡️ persuasion

Plan type

  1. Le langage permet pensée et communication.
  2. Mais il déforme : simplifie, manipule, crée des illusions.
  3. Donc : nécessité de clarifier les mots (définitions, critique, précision).
